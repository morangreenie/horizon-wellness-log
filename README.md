# Horizon Assembly Wellness Log

Static distribution for https://morangreenie.github.io/horizon-wellness-log/

The app stores wellness logs only in the current browser. It has no account,
server-side database, analytics, or automatic sharing. Clearing browser data or
using another browser starts a separate local record.

## Contents

- `index.html` and `404.html`: static app shell
- `assets/`: compiled application styles and scripts
- `og.png`: deterministic, code-rendered social preview
- `.nojekyll`: prevents Jekyll processing on GitHub Pages
